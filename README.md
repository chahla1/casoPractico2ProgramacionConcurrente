Esta aplicación Spring Boot realiza un benchmark de ejecución de tareas computacionalmente intensivas usando la ejecución secuencial, la ejecucion concurrente y la ejecucion asincrona con Spring.
El usuario puede configurar el número de tareas y el número de hilos mediante un endpoint REST.
